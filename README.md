# MedAIMedAI is a full-stack health triage web application built to make basic healthcare guidance more accessible. 
It features multiple specialized AI assistants (Medical, Emotional, Mental, and Physical Health), a Panic Mode 
with guided 4-7-8 breathing and grounding exercises, vitals monitoring, symptom history tracking, 
appointment booking, therapy finder, and a premium plans system.

Built with vanilla HTML/CSS/JS on the frontend (deployed on Vercel) and Node.js/Express/PostgreSQL/Prisma 
on the backend (targeting Render), with Google OAuth authentication, Gemini AI, and OpenRouter AI integration.
